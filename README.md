## Hi there! I'm Ryan May

I'm an indie game developer, software engineer, artist, and musician.

I'm the founder of Storykeeper Games, an independent studio creating fantasy games filled with rich worlds, memorable adventures, and clever mechanics. My passion lies at the intersection of technology and creativity, where programming, art, music, and design come together to create experiences players can get lost in.

As a solo creator, I enjoy every part of the process—from gameplay programming and tools development to 3D modeling, illustration, shaders, and composing music.

This GitHub is where I share that journey: game prototypes, developer tools, open-source projects, and the lessons learned while building games from the ground up.

## Philosophy

I enjoy understanding how things work beneath the surface. Whether that's building a web application, creating a game system, configuring a development environment, or experimenting with local AI models, I prefer learning the fundamentals rather than relying on black boxes.

My current long-term goal is to build and release indie games while continuing to grow as a software engineer.
